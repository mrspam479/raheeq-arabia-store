# رحيق Raheeq Arabia

Saudi-Arabian, Arabic-first, DTC functional-gummy DTC store. COD only. Built to convert KSA paid-social traffic (Snap / TikTok / Meta) at premium AOV.

This repository will contain two production services once the AI coder has finished scaffolding:

- `frontend/` — Next.js 15 + React 19 + TypeScript + Tailwind v4
- `backend/`  — Python 3.12 + FastAPI + SQLAlchemy 2 + Alembic + PostgreSQL

For now, the only authored artifact is the complete documentation under [`docs/`](./docs/).
